# 微信配置

## 添加账号

每个微信账号是一个独立的 iLink Bot API 连接。你可以添加多个账号。

在 Web 管理面板 → **WeChat** → **Add Account**：

- **ID**：账号唯一标识（如 `personal`、`work`），内部使用
- **Name**：显示名称（如「我的微信」、「工作号」）

## 登录

创建账号后，点击 **Login (QR)**：

1. Web 管理面板显示二维码
2. 打开手机微信 → 扫一扫
3. 手机上确认登录
4. 管理面板显示 "Connected"

登录会话会被保存。下次重启 ClawCenter 时，点击 **Connect** 即可恢复，无需重新扫码。

## 会话过期

微信会话会定期过期。过期时：

- ClawCenter 自动暂停轮询 1 小时，避免触发限流
- 暂停后自动恢复
- 如果持续失败，重新点击 **Login (QR)** 扫码登录

## 多账号

可以同时连接多个微信账号。每个账号：

- 有独立的登录会话
- 有独立的路由规则
- 维护独立的消息记录
- 运行独立的长轮询

适用场景：一个微信号用于个人，另一个用于团队，各自配置不同的 Agent。

## 消息同步

ClawCenter 使用长轮询（`getUpdates`）接收消息。同步位置保存在数据库中，重启服务不会丢失或重复消息。

## 断开连接

点击 **Disconnect** 停止接收消息，但不删除账号。点击 **Connect** 恢复。

点击 **Delete** 永久删除账号及其所有路由规则。
